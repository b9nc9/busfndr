# Használat
![header](https://raw.githubusercontent.com/b9nc9/busfndr/master/preview.png)
- Az [adat.py](https://github.com/b9nc9/volan)-n keresztül töltsd le a menetrendjeid, amelyeket az alkalmazásban szeretnél használni
- Az adatbázist másold át a telefonod tárhelyére
- Telepítsd fel az [alkalmazást](https://github.com/b9nc9/busfndr/releases/download/1.0/busfndr.apk), majd kövesd az utasításait
